Read all the BRDFs stored in a given directory.

Parameters
fileDirectorythe path of the directory where all the brdfs are stored
Returns
Z BRDFs data matrix where each column represents a BRDF

Initializes the list of BRDFs paths in the order in which they were read.

As the set of BRDFs can be heavy, we use the stxxl library Thus, a problem is not likely to occur if the RAM is too small compared to the set of BRDFs Indeed, in this case, the set of BRDFs is stored inside the disk

Read a BRDF from a file.

Parameters
filePaththe path of the BRDF's file
Returns
All the coefficients of a BRDF as a vector of scalars

If the file is not found, returns an error

As the set of BRDFs can be heavy, we use the stxxl library Thus, a problem is not likely to occur if the RAM is too small compared to the set of BRDFs Indeed, in this case, the set of BRDFs is stored inside the disk
