Detailed Description

template<typename Scalar>
class ChefDevr::OptimisationSolver< Scalar >

Class that solves the optimisation problem defined in the research paper: A Versatile Parametrisation for Measured Materials Manifold. Computes an optimised mapping from BRDFs space to a latent space.

Template Parameters
ScalarThe type of scalar number to do computations with. The precision of this type is crucial to reconstruct an accurate BRDF from the latent space

Computes the new inverse matrix K_minus1 and the new determinant of K using Sherman-Morisson formula.

Parameters
old_K_minus1K_minus1 matrix before K had changed
new_K_minus1K_minus1 matrix after K has changed (computed in this function)
old_detKDeterminant of K before K had changed
new_detKDeterminant of K after K has changed (computed in this function)
lv_numNumber of the latent variable that has changed
diff_cov_vectorThe difference between the column/row of K that has changed and the column/row before it had change. Although this parameter is not const it remains unchanged when the function returns
Precondition
old_detK != 0

Computes new_K_minus_1 and new_detK from their previous values using Sherman-Morisson formula. The formula for the inverse and determinant are applied twice : one for the row modification, and another one for the column modification.
